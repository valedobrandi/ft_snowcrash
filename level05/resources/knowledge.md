The script /usr/sbin/openarenaserver is owned by flag05 and executes every file located in /opt/openarenaserver/. It loops through the directory and runs each file with bash, then deletes it afterward. Because this directory is writable by the attacker, it is possible to place a malicious script inside it. The attacker creates a script containing getflag > /tmp/flag05 and saves it as /opt/openarenaserver/getflag.sh. When the privileged script runs, it executes this file with the permissions of flag05. This runs the getflag command and writes the result to /tmp/flag05. The attacker then repeatedly checks /tmp/flag05 until the flag appears. This exploit works because the privileged script executes user-controlled files without validation.
